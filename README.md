# ReactNativeDemo app

ReactNativeDemo is an event countdown app. It shows how much time is to a set date.

## Installation

Use the package manager `npm` to install ReactNativeDemo and `json-server` for the backend. 

```bash
cd ReactNativeDemo
npm install
```

## Usage

```bash
npm run ios
npm run android
```

```bash
json-server db.json
```
